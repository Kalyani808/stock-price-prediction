# 📈 Stock Price Prediction using Machine Learning

## 🚀 Overview

This project predicts future stock prices using historical data and a Linear Regression model.

## 🧠 Concepts Used

* Machine Learning (Linear Regression)
* Time Series Data
* Data Preprocessing
* Model Evaluation (R² Score)

## 🛠️ Technologies

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib
* yfinance

## 📊 Features

* Fetches real-time stock data
* Predicts future prices (30 days ahead)
* Visualizes actual vs predicted prices
* Calculates model accuracy

## 📷 Output

* prediction_plot.png

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
```
